# PRD：古诗研习录 · 单文件 HTML 学习应用

## 1. 项目概述

- **产品名称**：古诗研习录（或"诗境"）
- **形态**：单文件 HTML（`.html`），所有代码、数据、样式内嵌，零后端，零依赖构建
- **目标**：让用户按 4 个 Level 循序渐进学习 75 首小学必背古诗，记录"理解 / 手写 / 背诵"三维进度
- **风格关键词**：**新中式萌系**（国风可爱）—— 宣纸纹理、朱砂印章、卷轴卡片、大圆角、柔和弥散阴影、手绘感图标

---

## 2. 用户旅程与页面结构

```
[首页] → [Level 选择页] → [诗列表页] → [诗详情学习页]
 ↓           ↓                ↓
[统计面板]  [复习模式]      [随机抽背]
```

### 2.1 页面清单

| 页面 | 说明 |
|------|------|
| **首页** | 欢迎语 + 总进度圆环 + 4 个 Level 入口卡片 + 今日推荐 + 底部导航 |
| **Level 页** | 该 Level 的诗列表，每首显示缩略信息 + 三维进度小图标 |
| **详情页** | 单首诗的完整学习界面：原文大字、拼音（可选）、注释、译文、背诵技巧、操作区 |
| **统计页** | 总进度、Level 进度条、已学/待学数量、连续学习天数（基于 localStorage 首次打开日期计算） |
| **复习页** | 只显示已标记"已背诵"的诗，支持随机抽背（只显示标题，点击展开原文自测） |

## 3. 视觉设计系统（Design System）

### 3.1 色彩

```css
:root {
  --paper: #F7F3E9;           /* 宣纸底色 */
  --paper-dark: #EDE8DA;      /* 卡片底色 */
  --ink: #2C2C2C;             /* 墨黑，主文字 */
  --ink-light: #5A5A5A;       /* 次要文字 */
  --cinnabar: #D44A4A;        /* 朱砂红，强调、印章、完成态 */
  --cinnabar-light: #F5E6E6;  /* 朱砂淡底 */
  --bamboo: #6B9E75;          /* 竹青，进行中、理解标记 */
  --bamboo-light: #E8F5E9;    /* 竹青淡底 */
  --gold: #C9A86C;            /* 淡金，装饰、Level 4 */
  --gold-light: #F5F0E6;      /* 淡金底色 */
}
```

### 3.2 字体

- **标题/诗句**：`"LXGW WenKai", "ZCOOL XiaoWei", "STKaiti", serif`（通过 CDN 引入霞鹜文楷）
- **正文/注释**：`"LXGW WenKai", "PingFang SC", "Microsoft YaHei", sans-serif`
- **拼音**：`"Arial", sans-serif`（小字号，灰色）

> **字体 CDN**：`https://cdn.jsdelivr.net/npm/lxgw-wenkai-webfont@1.7.0/style.css`

### 3.3 组件风格

- **卡片**：背景 `--paper-dark`，圆角 `20px`，阴影 `0 8px 32px rgba(44,44,44,0.08)`，边框 `1px solid rgba(0,0,0,0.04)`
- **按钮**：圆角 `12px`，主按钮用 `--cinnabar` 白字，次按钮用 `--paper` 底色 + `--ink` 边框
- **印章**：完成标记用正方形圆角小章，红底白字，旋转 `-6deg`，字体用粗体楷体
- **卷轴装饰**：顶部/底部用细线 + 小圆点模拟卷轴杆
- **吉祥物**：页面角落放一个手绘风格小熊猫（CSS/SVG 绘制，简单圆形+耳朵+竹叶），点击可播放鼓励语音（可选，可用 Web Speech API 读当前诗句）

### 3.4 背景

- 全局背景：`--paper` 底色 + 微妙的宣纸纹理（base64 内联 SVG 或 CSS 渐变模拟）
- 水墨点缀：页面切换时，角落有淡墨圆晕扩散动画（CSS `radial-gradient` + `animation`）

## 5. 功能需求详解

### 5.1 首页

- **顶部**：应用名"古诗研习录" + 小熊猫图标
- **总进度圆环**：SVG 绘制，显示 75 首中"已背诵"的比例，中间数字大写
- **Level 入口**：4 张卡片横向排列（移动端 2×2）
  - 每张显示：Level 名、副标题、该 Level 完成度（小进度条）、锁定状态（Level N 需 Level N-1 背诵完成 50% 解锁，或全部开放，由用户选择，**默认全部开放**）
  - 卡片颜色区分：
    - L1: `--cinnabar-light` 底 + `--cinnabar` 强调
    - L2: `--bamboo-light` 底 + `--bamboo` 强调
    - L3: `--gold-light` 底 + `--gold` 强调
    - L4: 淡紫 `#F3E8F5` 底 + `#8E6B8C` 强调
- **今日推荐**：随机推荐一首未背诵的诗，点击直达详情
- **底部导航**：首页 / 统计 / 复习（图标 + 文字）

### 5.2 Level 页

- 顶部返回 + Level 标题（如"Level 1 · 零基础热身"）+ 该 Level 进度
- 诗列表：垂直排列，每行一首诗
  - 左侧：序号（圆形底色）
  - 中间：诗名 + 作者 + 朝代（小标签）
  - 右侧：三个状态小圆点（理解/手写/背诵），完成的点亮，用不同颜色
  - 已完成的诗，诗名加 `--cinnabar` 色，右侧盖小印章"已会"
- 点击整行进入详情页

### 5.3 诗详情页（核心页面）

布局从上到下：

1. **卷轴头**：装饰性横条
2. **标题区**：诗名大字（36px）+ 作者/朝代标签
3. **原文区**：
   - 每句一行，居中，大字（28px），字间距宽松
   - 句末有"朗读"小按钮（Web Speech API 朗读该句）
   - 点击单句可高亮
4. **注释区**（折叠面板，默认展开）：
   - 关键词 + 解释，用 `--gold-light` 底色小卡片
5. **译文区**（折叠面板，默认展开）：
   - 现代汉语翻译
6. **背诵技巧区**（折叠面板，默认展开）：
   - 用 `--bamboo-light` 底色，带小灯泡图标
7. **操作区**（固定在底部或紧跟内容）：
   - 三个大按钮：
     - "我理解了"（切换 understood）
     - "已手写"（切换 handwritten）
     - "已背下"（切换 recited）—— 点击后有**撒花/盖章动画**
   - 按钮状态：未完成时白底描边，完成后填充色 + 打勾
8. **导航**：上一首 / 下一首 / 返回列表

### 5.4 统计页

- 顶部：学习天数、连续天数
- 中部：4 个 Level 的进度条
- 下部：环形图或条形图（用 CSS 绘制，不依赖图表库）显示：
  - 已理解 / 已手写 / 已背诵 的数量
  - 待学习数量
- 底部：重置进度按钮（二次确认）

### 5.5 复习页

- 模式选择：顺序复习 / 随机抽背
- **抽背模式**：
  - 只显示诗名 + 作者
  - 用户在心里背诵，点击"展开原文"核对
  - 核对后标记"记得"或"模糊"或"忘记"
  - "模糊/忘记"的诗加入"需加强"列表

## 6. 交互与动画

| 场景 | 动画 |
|------|------|
| 页面切换 | 淡入 + 轻微上移（`opacity` + `translateY`），300ms，ease-out |
| 标记完成 | 按钮点击后，从按钮位置爆发小粒子（CSS 伪元素 + `animation`），同时盖朱砂印章 |
| 诗列表加载 | 逐行 stagger 淡入，每项延迟 50ms |
| 朗读时 | 被朗读的句子高亮（`background-color` 过渡） |
| 滚动 | 顶部出现细卷轴进度条（`position: fixed`） |
| 成就达成 | 当完成一整个 Level 时，全屏淡入"Level 通关"横幅，带 confetti（用 CSS 绘制彩色小方块下落） |

---

## 7. 技术栈

- **框架**：无，原生 HTML5 + CSS3 + ES6+
- **样式**：Tailwind CSS v3（CDN：`https://cdn.tailwindcss.com`）
- **字体**：霞鹜文楷（LXGW WenKai）CDN
- **存储**：localStorage（无后端）
- **语音**：Web Speech API（`speechSynthesis`）用于朗读
- **图标**：内联 SVG（手绘风格，不引入外部图标库）

---

## 8. 响应式断点

- **Mobile**（< 640px）：单列，底部固定导航，卡片全宽
- **Tablet**（640-1024px）：Level 卡片 2×2，列表可双列
- **Desktop**（> 1024px）：居中容器最大宽度 800px，详情页左右留白，卷轴效果更明显

---

## 9. 单文件约束说明

- 所有 CSS 写在 `<style>` 中，不使用外部 CSS 文件
- 所有 JS 写在 `<script>` 中，不使用外部 JS 文件
- 所有图片用 CSS 绘制、SVG 内联、或 base64（如需要宣纸纹理）
- 数据直接内嵌为 `const POEMS_DATA = [...]`
- 使用 Tailwind CDN 时，需在 script 中配置 theme 扩展自定义颜色
